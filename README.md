# 🌟 To-Do List Application

A colorful and responsive To-Do List application built using HTML, CSS, JavaScript and localStorage.

## Features

- ✅ Add Task
- ✏️ Edit Task
- 🗑 Delete Task
- ✔ Mark Completed
- 📋 Task Counter
- 🕒 Live Date & Time
- 💾 localStorage Support
- 🔍 Filter Tasks (All, Active, Completed)
- 🎨 Responsive Colorful UI

## Technologies Used

- HTML
- CSS
- JavaScript
- localStorage
